Recently I came up with an idea for an improved type system. I call it predicate typing. I can guarentee you that I am absolutely not the first person to come up with an idea like this. I actually imagine that many programmers think about code this way when they first start out. At least I did.

After forming the idea I began to look around for something similar and found predicate dispatch, which seems very familiar. However, as of right now I'm not fully convinced that it is exactly the same.

So, what is predicate typing?

It is no more restriction than anything that we naturally put on stuff.
